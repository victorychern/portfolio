Telegram Mini App для покупки прокси

ProxyJam Mini App — мобильное Telegram-приложение для покупки прокси и управления ими. Продолжение существующего веб-сервиса ProxyJam, адаптированное под Telegram с учётом ограничений платформы.

01.

О проекте

Разработала дизайн мобильного Telegram Mini App для ProxyJam — сервиса по продаже прокси. Спроектировала сценарии выбора типа прокси, оформления заказа с оплатой через Telegram Stars, управления списком прокси с фильтрацией и поиском, а также экраны профиля с историей заказов. Создала UI-кит и переработала визуальный стиль с нуля, заложив поддержку светлой и тёмной тем.

04.

Почему не просто адаптация веб-версии

При разработке Mini App стало очевидно, что прямая адаптация существующего веб-дизайна невозможна по ряду причин.

Отсутствие поддержки тёмной темы. Дизайн-система веб-версии строилась под светлый фон — цвета, тени и компоненты не предполагали инверсии. Telegram же по умолчанию поддерживает тёмную тему на уровне платформы, и пользователи ожидают, что приложение следует системным настройкам. Переписать существующий UI-кит под оба режима оказалось бы дороже, чем создать новый с нуля с поддержкой токенов для Light/Dark.

Несовместимость компонентной базы. Веб-компоненты (модальные окна, таблицы, выпадающие списки) плохо переносятся на мобильный экран: таблица с прокси из веба при переносе «один в один» становилась нечитаемой на 375px. Потребовался полный пересмотр паттернов в сторону нативных мобильных: bottom sheets, swipe-действия, floating tab bar.

Новая навигационная логика. В мини-приложении нет полноценного браузерного back-button и сайдбаров, характерных для десктопа. Пришлось проектировать навигацию заново под iOS-паттерны внутри WebView.

Изменение платёжной механики. Веб-версия работала с классическими способами оплаты. Mini App ориентирован на Telegram Stars — это потребовало переработки экранов заказа.

05.

Ограничения и сложности

Ограничения Telegram WebView. Mini App работает в WebView внутри мессенджера, что накладывает технические ограничения: специфика отображения, жёсткая зависимость от нативных элементов Telegram (header, кнопки закрытия/навигации). Часть привычных веб-паттернов просто не работала.

Согласованность с веб-версией. Приложение существует параллельно с веб-версией, поэтому несмотря на обновлённый визуал, логика сценариев и терминология должны были оставаться узнаваемыми для пользователей, которые уже знакомы с сервисом.

Неопределённость по провайдерам. Часть данных (список провайдеров, доступных стран) на момент работы ещё не была окончательно определена на бэкенде, поэтому часть полей проектировалась с расчётом на динамические данные.

Платёжный сценарий через Stars. Telegram Stars — относительно новый платёжный инструмент, и нужно было спроектировать понятный UI для пользователей, которые могут не знать, как работает этот механизм оплаты.

06.

Дизайн

Главный экран

Главный экран отличается для нового и активного пользователя. Новому показывается промо-баннер с промокодом на первый заказ и карточка с нулевыми прокси с призывом сделать заказ. Активному — карточки с текущими прокси, счётчики использования трафика и интерактивный график потребления по дням, неделям или месяцам.

Выбор типа прокси

При нажатии «Order proxy» открывается экран выбора типа прокси. Каждый тип оформлен как отдельная карточка с иллюстрацией и описанием, свайп переключает между ними. Ниже — список ключевых преимуществ и кнопка оформления заказа.

Форма заказа

Форма заказа собрана на одном экране: тип прокси в шапке, параметры (страна, город, провайдер, период) через bottom sheets, переключатель автопродления, цена в Telegram Stars и слайдер количества IP. Всё сделано в один шаг — без дополнительных страниц и редиректов.

Список прокси

Экран списка прокси разбит по вкладкам типов (Residential ISP / Rotating Mobile). Сверху — фильтр по статусу (All / Active / Expires / Inactive), поиск и кнопка фильтров. Каждая строка показывает IP:порт, флаг страны, статус и протокол. Справа — тоггл авто-продления.

Страница поиска (найдено по фильтру)

При применении фильтра интерфейс отображает количество найденных результатов с возможностью сбросить фильтр. Поиск работает поверх всего содержимого таблицы.

Профиль

Профиль отображает Telegram-никнейм пользователя и аватар, взятый из аккаунта. Настройки минимальные — история заказов, смена темы и языка. Переключение темы реализовано через bottom sheet с тремя вариантами: Light / Dark / System.

Светлая и тёмная тема

Все экраны проработаны в двух темах. Для тёмной темы переработана вся цветовая схема: фоны, карточки, иконки, графики. Граф трафика в тёмной теме использует более насыщенные акцентные цвета на тёмном фоне, что делает данные более читаемыми.
